  3. Container :


    4. Container tags are the tags that contain some data such as text, image, etc. There are several container tags in HTML.

    <div> Tag


    div tag is used to make blocks or divisions in the document. 
    <div> This is div block </div>

    <p>Tag


    p tag is used to createa Paragraph in the document. 
    <p> This is a Paragraph </p>

    <pre> Tag


    pre tag is represents pre-formatted text in the document. 
    <pre> Hello World </pre>

    <code> Tag


    code tag is used to represent source codes in the document. 
    <code> import python </code>


  4. Text Formatting :


    5. Text formatting tags are used to format text or data of HTML documents. You can do certain things like creating italic, bold, strong text to make your document look more attractive and understandable.

    <b> Tag 

    <b> Write here Bold text </b>

    <i> Tag 

    <i> Write here italic text </i>

    <u> Tag 

    <u> Used tag for Underline </u>

    <strong> Tag 

    <strong> Write here important text </strong>

    <em> Tag 

    <em> Write here Emphasized text </em>

    <sub> Tag 

    <sub> Subscript </sub>

    <sup> Tag 

    <sup> Superscript </sup>

    <br> Tag 

    <br> # it breaks the line of code


  5. Lists :


    6. Lists can be either numerical, alphabetic, bullet, or other symbols. You can specify list type and list items in HTML for the clean document.

    <ol> Tag


    Ordered list starts with <ol> tag and each list item starts with <li> tag 
    <ol>
    <li> Data 1 </li>
    <li> Data 2 </li>
    <li> Data 3 </li>
</ol>

    <ul> Tag 

    <ul>
    <li>Your Data</li>
    <li>Your Data</li>
</ul>

    <li> Tag 

    <li> Write here list </li>


  6. Media :


    7. Media is anything that is present in digital form such as image, video, audio, etc.

    <img> Tag


    It is used to embed or import image in a webpage. 
    <img src="Source_of_image" alt="Alternate text">

    <video> Tag


    It is used to embed video in the webpage. 
    <video width="480" height="320" controls>
    <source src="demo_move.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

    <audio> Tag


    It is used to embed sound content in the document. 
    <audio controls>
    <source src="demo.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

  12. Semantic Elements :


    13. Semantic elements are those elements that are self describable, i.e., from their name itself, you can understand their meaning.

    <section> Tag


    It represents a section in the document 
    <section> This is a section </section>

    <article> Tag


    It represents self-contained content 
    <article> Enter your data here </article>

    <aside> Tag


    It is used to place content in the sidebar 
    <aside> Enter your data here </aside>
